# 블록체인 기반 전자투표 시스템

> 탈중앙화 · 무결성 · 익명성 · 접근성을 갖춘 온라인 투표 플랫폼

---

## 💡 프로젝트 소개

선거 조작 및 부정선거 논란으로  
“내 표가 올바르게 반영되었는가?”에 대한 신뢰 문제가 전 세계적으로 제기되고 있다.

이 프로젝트는 **블록체인 기반 전자투표 시스템**을 구축하여

- 누구나 검증 가능한 투표 기록
- 특정 기관에 의존하지 않는 투표 신뢰

을 실현하는 것을 목표로 한다.

---

## 🎯 목표

- 조작이 어려운 투표 시스템 구축
- 투표 결과의 검증 가능성 제공
- 사용자 친화적인 UI 및 접근성 제공
- 사회적/기술적 신뢰 동시 확보

---

## 🔑 핵심 가치

- **탈중앙화(Decentralization)**  
  중앙 서버 신뢰 의존 최소화

- **무결성(Integrity)**  
  투표 기록 위·변조 불가

- **익명성(Anonymity)**  
  신원 인증과 투표 데이터를 분리하여 프라이버시 보호

- **접근성(Accessibility)**  
  웹 기반 플랫폼을 통한 쉬운 투표 경험 제공

---

## 🛠 주요 기능

### ✅ 1. 유권자 인증 및 로그인

사용자가 **2가지 로그인 방식 중 선택 가능**

✔ **DID 로그인**

- 고유 DID 기반 신원 검증
- DID는 투표 행위와 분리 저장되어 익명성 유지

✔ **MetaMask 로그인**

- 지갑 주소 기반 인증
- 트랜잭션 서명 및 투표 확정에 사용

➡ DID/지갑 주소는 **투표 행위와 연결되지 않아 익명성 보장됨**

---

### ✅ 2. 투표 생성 및 참여

✔ 관리자 UI에서 투표 생성  
✔ 후보 등록 및 기간 설정  
✔ 관리 암호 복잡도 정책(소문자+숫자+특수문자) 적용  
✔ 투표 참여 → 결과 집계 → 기록 검증 UI 제공

---

### ✅ 3. 블록체인 기록 및 검증

- 투표 생성, 참여 등 주요 이벤트를 스마트 컨트랙트에 기록
- 결과는 블록 익스플로러(예: Sepolia Explorer 등)로 검증 가능

---

### ✅ 4. 보안 강화

- 관리자 삭제/수정 기능은 **비밀번호 + DID/지갑 인증 요구**
- 투표 데이터는 블록체인에 기록되어 위·변조 불가
- 로그인 방식을 분리하여 익명성 유지 & 접근성 강화

---

### ✅ 5. 모의 선거 / 시뮬레이션

- 학습·연구용 데모 시나리오 지원
- 기존 방식 대비 비교 테스트 가능

---

## 🧩 시스템 아키텍처

### 🔷 Frontend Layer

- React + Next.js 기반 UI
- TypeScript, Tailwind CSS
- 로그인 방식 선택 화면 (MetaMask / DID 중 선택)

### 🔷 Identity Layer

- DID 기반 신원체계
- MetaMask 지갑 인증 지원
- 두 인증 방식 모두 **투표 데이터와 분리되므로 익명성 유지**

### 🔷 Smart Contract Layer

- Solidity 기반 투표·집계 로직
- 투표 생성/참여 트랜잭션 처리
- EVM 네트워크(Sepolia 등) 연동

### 🔷 Storage / Backend Layer

- Supabase 및 로컬 저장소
- 투표 메타데이터 저장 및 조회

- 내부 프로세스가 일반 사용자에 불투명

✔ **블록체인 기록 기반 검증 가능성 제공**

✔ 투표 생성부터 검증까지 과정을 UI로 시각화

= “보여지는 신뢰 + 설명 가능한 신뢰”
